# 健身动作网站

基于 FastAPI 的电脑端健身动作展示与智能助手网站，内置 1,324 条健身动作数据，支持分页浏览、多维度筛选、动作详情查看，并为每个动作配备独立的 SSE 流式多轮健身助手。

## 功能特性

- **动作浏览**：首页分页展示全部 1,324 条健身动作，每页 24 条
- **多维筛选**：支持动作分类、目标肌肉、器材三个维度单选筛选及任意组合筛选
- **详情查看**：点击动作卡片弹出详情弹窗，展示 GIF 动画及完整动作数据
- **智能助手**：每个动作附带独立的多轮对话窗口，采用 SSE 流式回复
- **上下文记忆**：首次对话自动发送完整动作数据作为上下文，后续对话保留历史记录，避免重复传输

## 技术栈

- **后端框架**：FastAPI + Uvicorn
- **前端渲染**：Jinja2 模板 + 原生 JavaScript
- **HTTP 客户端**：httpx（异步流式请求）
- **数据格式**：JSON
- **包管理**：uv
- **测试框架**：pytest

## 安装与启动

### 环境要求

- Python 3.13+
- uv 包管理器

### 安装步骤

1. 克隆项目到本地
2. 在项目根目录执行依赖同步：

```bash
uv sync
```

### 配置说明

项目运行需要配置 `LONGCAT_API_KEY` 环境变量。该密钥必须由操作系统预先配置，启动脚本会自动读取。

### macOS 启动

```bash
chmod +x start.sh
./start.sh
```

启动脚本会自动检查 `LONGCAT_API_KEY` 是否已配置，若缺失将显示中文错误提示并退出。检查通过后执行 `uv sync` 同步依赖，最后启动 Uvicorn 服务。

### Windows 启动

双击 `start.bat` 或在命令行执行：

```bash
start.bat
```

启动脚本会自动检查 `LONGCAT_API_KEY` 是否已配置，若缺失将显示中文错误提示并退出。检查通过后执行 `uv sync` 同步依赖，最后启动 Uvicorn 服务。

### 访问服务

启动成功后，打开浏览器访问：

```
http://localhost:8000
```

## 使用说明

1. **浏览动作**：首页以网格形式展示动作卡片，底部支持分页导航
2. **筛选动作**：使用顶部下拉框选择分类、目标肌肉或器材，支持组合筛选
3. **查看详情**：点击任意动作卡片，弹出详情窗口展示 GIF 动画、动作名称、分类、目标肌肉、使用器材及训练步骤
4. **智能对话**：在详情窗口底部打开健身助手，输入问题获取专业建议。首次提问会自动附带当前动作完整信息，后续对话保留上下文

## 数据说明

- 数据来源：`exercises-dataset/data/exercises_zh.json`
- 动作总数：1,324 条
- 每条动作包含唯一 ID、名称、分类、目标肌肉、肌肉群、器材、训练步骤、缩略图及 GIF 动画
- 媒体资源（图片/GIF）归 Gym visual 所有，仅用于展示

## 测试验证

项目包含后端接口测试，覆盖分页筛选、详情查询、SSE 对话流式响应等核心功能。执行以下命令运行测试：

```bash
uv run pytest -q
```

测试通过表示服务核心逻辑运行正常。

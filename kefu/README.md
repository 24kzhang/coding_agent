# 手机销售店铺智能客服系统

## 项目简介

本项目是一个面向手机销售店铺的智能客服系统，提供商品咨询、价格推荐和售后问答等功能。系统采用轻量级Web应用架构，支持中文对话交互。

## 主要功能

### 1. 商品咨询
- 支持手机型号、配置、颜色等基础信息查询
- 提供手机参数对比功能
- 支持多轮对话，理解用户意图

### 2. 价格推荐
- 根据用户预算推荐合适的手机型号
- 支持价格区间筛选
- 提供性价比分析和购买建议

### 3. 售后问答
- 常见问题自动解答
- 退换货政策查询
- 保修服务说明

## 启动方式

### 方式一：直接打开
双击 `index.html` 文件，使用浏览器打开即可运行。

### 方式二：本地服务器（推荐）
```bash
# 进入项目目录
cd kefu

# 使用Python启动简单HTTP服务器
python -m http.server 8000

# 或使用Node.js的http-server
npx http-server
```

启动后访问 `http://localhost:8000` 即可使用。

> **注意**：由于浏览器安全策略限制，建议使用本地服务器方式打开，避免 fetch 加载 JSON 数据时出现跨域问题。
